1. **Setup**: Start with [setup.md](setup.md) to configure your environment
2. **Theory**: Read [railway-optimization.md](railway-optimization.md) for MILP and network flow fundamentals
3. **Practice**: Run the examples in `src/optimization/` to see routing in action
4. **Data**: Learn [data-workflow.md](data-workflow.md) for managing datasets
5. **Analysis**: Use [notebook-guide.md](notebook-guide.md) for exploratory work
